SLAYER: downloads (downloads)

The official releases of SLAYER are available at ftp://download.gnu.org.ua/release/slayer . Apart from the source code, the self-contained build for the Windows operating system can be obtained from there as well (it includes all necessary runtime libraries).

Note however, that since SLAYER is a relatively young and quickly growing project, the official tarballs don't necessarily contain all the latest improvements, which can be found in the Mercurial repository:

  hg clone http://hg.gnu.org.ua/hgweb/slayer

The repository is also mirrored by the bitbucket service:

  hg clone https://bitbucket.org/panicz/slayer

Both services provide a facility to browse the source code online.
