SLAYER: screenshots (demo screenshots)


This is a screenshot of a clone of Pandora Game written by drcz for the 3rd birthday of GNU Guile 2.0, with images credited to johann-c. The source code can be viewed in the repository (the demos/pandora.scm file). The goal of the game is to shut down some reactors using a robot that is running out of energy.


Above this text one can see the a screenshot from Schess, an implementation of Chess written in Scheme. Schess is particularly notable for the way the game rules are written (which actually makes it a framework for describing many chess-alike games).


The image above is a screenshot from the original SLAYER demo. It allows you to move the camera in a 3d space (using mouse and WSAD keys), generate new objects (by right-clicking on the existing ones) and drag them around the screen and rotate them around their centers. There's also an instance of <text-area> class, which allows to edit text and evaluate Scheme expressions in it.


This one demonstrates a physics simulation performed by the Open Dynamics Engine. There are ODE wrappers for Guile availabie in SLAYER repository, as well as a tool for converting the Stanford PLY format to Slayer's 3D format.


There's also an implementation of the classical game of PONG


a and a functional implementation of Snake, written by Jens Nicolay.
